# How to Run the Project

1. Install XAMPP
2. Copy project folder into:

```
htdocs
```

3. Start Apache and MySQL in XAMPP

4. Create database in phpMyAdmin:

```
vdstore
```

5. Run project in browser:

```
http://localhost/vdstore/index.html
```
